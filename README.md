Сайт кафе, разработанный на **Wagtail CMS** (Django).

## Технологии
- Wagtail 6.x
- Django 6.x
- SQLite

## Особенности
- Адаптивный дизайн
- Галерея изображений через slick-carousel
- Меню с категориями
- Форма обратной связи
