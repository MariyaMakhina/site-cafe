## 🎯 Реализованный функционал

- **Карусель меню** — на чистом JavaScript с автолистанием и ручным переключением
- **Форма бронирования стола** — отправка заявок на почту и в Telegram
- **Адаптивная вёрстка** — корректное отображение на всех устройствах (ПК, планшет, телефон)
- **Админ-панель Wagtail (StreamField)** — клиент сам управляет блюдами, ценами и фотографиями в карусели

## 🛠 Стек технологий

- Python 3.11, Django, Wagtail CMS
- PostgreSQL
- HTML, CSS, JavaScript
- Git
